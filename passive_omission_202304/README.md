# Usage

Run on windows command line window:
```
python run_postprocess.py `
--session_name 'FN15_P_omi_032124_w' `
--range_skew '0,5' `
--max_connect '3' `
--range_footprint '1,3' `
```

Run on linux server:
```
sbatch run_postprocess.sh
```
